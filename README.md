# ISOBreaker

A tool to read and extract from various ISO and BIN/CUE Formats.

## Currently Supported ISO Formats

Uncompressed ISO

## Planned ISO Formats
GDI, CDI, CHD, CSO, WIA and RVZ (Gamecube Only)

## Supported ISO Filesystems
ISO 9660, Joilet, Gamecube, Xbox

## Planned ISO Filesystems
UDF, NitroFS

## Note
Wii ISO's are encrypted and for legal reasons, there are no plans to support them in the program.
